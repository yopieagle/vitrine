B) Color Schema

Artemis comes with 6 predefined colors orange, green, blue, turquoise, yellow and grey.

You can change the color of theme by the following code in index.html, <head> tag: 

<link class="colors" rel="stylesheet" href="layout/colors/orange.css" type="text/css">

D) Twitter Feed

The template comes with a twitter feed section to display the last tweet from you twitter account. By default, it's set the twitter feed the display my own twitter feed for the preview purposes. To change the twitter feed to display your own feed, open layout/js/main.js and go to line 740. You'll find this line: 

username:'flashblue80', //Change username

You only have to change the username to yours.

Important note: Since Twitter requires every request to be authenticated due the new version 1.1 of twitter API, it needs some private informations and settings. I recommend that you open the index.php file in the twitter folder and fill in with your own API settings. You can create a Twitter app here: https://dev.twitter.com.

The index.php is located in the layout/plugins/twitter/ folder. If you open it there is a config section from line 24 you are only interested in this four lines below:

// Your Twitter App Consumer Key
private $consumer_key    = '';

// Your Twitter App Consumer Secret
private $consumer_secret = '';

// Your Twitter App Access Token
private $user_token      = '';

// Your Twitter App Access Token Secret
private $user_secret     = '';

You have to change this information to yours, when you create your twitter app here https://dev.twitter.com/ you get this information under My applications tab.

E) Contact Form

You need to configure contact.php to get e-mails from the users.

Change those variables on line 11 with the real name & e-mail address to send the mail information:

$toName = "Artemis";
$toAddress = "email@sitename.com";
